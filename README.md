# Dysfunctional Family

A Mario-style side-scrolling platformer built as a single HTML5 Canvas file with zero external libraries. Works on any phone browser.

## Play

Open `index.html` in any modern browser, or play the hosted version on GitHub Pages.

## Features

- **Character Select** — Choose from 8 family members, each with a unique RPG class sprite (Archer, Knight, Rogue, Warrior, Ninja, Swordsman, Mage, Ranger)
- **Enemy Family** — The family members you didn't pick become enemies who fire angry emoji bombs and hurl insults at you
- **Power-ups** — Collect palm tree power-ups for Shield (invincibility), Speed boost, or Coin Magnet
- **Coin Collection** — Spinning gold coins with sparkle effects scattered across platforms
- **Parallax Background** — Multi-layer night sky with mountains, treeline, clouds, moon, and twinkling stars
- **Mobile Controls** — On-screen d-pad and jump button for touch devices
- **Keyboard Support** — Arrow keys / WASD + Space to jump
- **Zero Dependencies** — Single HTML file, no libraries, no external images — everything drawn with Canvas 2D API
